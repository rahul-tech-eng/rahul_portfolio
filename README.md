# Rahul Chand Portfolio Website

## Overview

A personal portfolio website showcasing my skills, education, projects, certifications, and contact information. This website was built to present my work, technical skills, and achievements as a Computer Science student and aspiring Web Developer.

## Features

* Responsive design for desktop and mobile devices
* Modern and professional user interface
* About Me section
* Education section
* Technical Skills showcase
* Projects section
* Certifications section
* Contact section
* Downloadable résumé

## Technologies Used

* HTML5
* CSS3
* JavaScript
